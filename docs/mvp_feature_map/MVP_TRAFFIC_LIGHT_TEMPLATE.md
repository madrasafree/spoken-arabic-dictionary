<h1>תבנית רמזור לתעדוף MVP (ירוק / צהוב / אדום)</h1>

<p><strong>תאריך עדכון:</strong> __________</p>
<p>מטרת התבנית: למלא בצורה אחידה את מצב המוכנות של כל פיצ'ר לקראת הגרסה החדשה.</p>

<h2>הגדרת צבעים</h2>
<ul>
  <li><strong>ירוק:</strong> מוכן ל-MVP עכשיו (מסך, נתונים, UX בסיסי ובדיקות עיקריות).</li>
  <li><strong>צהוב:</strong> עובד חלקית, יש פער ממוקד לפני כניסה ל-MVP.</li>
  <li><strong>אדום:</strong> לא בסקופ MVP כרגע או דורש עבודה מהותית לפני שילוב.</li>
</ul>

<h2>כללי מילוי קצרים</h2>
<ul>
  <li>שורה אחת לכל פיצ'ר.</li>
  <li>לצרף הוכחה אחת לפחות (URL, צילום, דמו, PR).</li>
  <li>אם הפיצ'ר צהוב או אדום: לציין פעולה הבאה ברורה.</li>
</ul>
