# תבנית רמזור לתעדוף MVP (ירוק / צהוב / אדום)

תאריך עדכון: __________

מטרת התבנית: למלא בצורה אחידה מצב מוכנות לכל פיצ'ר לקראת MVP של הגרסה החדשה.

## הגדרת צבעים

- ירוק: מוכן ל-MVP עכשיו (מסך + נתונים + UX בסיסי + בדיקות עיקריות).
- צהוב: עובד חלקית, חסרים פערים ממוקדים לפני כניסה ל-MVP.
- אדום: לא בסקופ MVP כרגע או דורש עבודה מהותית לפני שילוב.

## כללי מילוי קצרים

- למלא שורה אחת לכל פיצ'ר.
- לצרף הוכחה אחת לפחות (URL, צילום, דמו, PR).
- אם צהוב/אדום: לציין פעולה הבאה ברורה שאפשר לבצע.
